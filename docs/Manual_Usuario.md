# Manual de Usuario

## Acceso
- Inicia sesión con tu cuenta. Si no tienes, solicita registro.

## Panel Admin
- Dashboard: KPIs y tendencias.
- Productos: listado con búsqueda, orden y paginación. Acciones de editar/eliminar.
- Clientes: listado con búsqueda, orden y paginación.
- Ventas: listado con filtros por texto/fechas, orden y paginación.

## Búsqueda y filtros
- Barra de búsqueda para filtrar por texto.
- Enlaces de orden (Fecha/Precio/Stock/Nombre) ajustan el orden.
- “Por página” ajusta cantidad mostrada (cuando está disponible).

## Carga de imágenes
- En formularios compatibles, sube una imagen; se guarda en `public/uploads`.

## Notificaciones
- Acciones muestran toasts en la esquina superior derecha.
